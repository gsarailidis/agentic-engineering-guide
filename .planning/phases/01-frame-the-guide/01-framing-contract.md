# Phase 1 Framing Contract

## Scope Statement

The guide is about local tool-using agents operating inside local execution environments, not agentic AI in general.

## Example Strategy

Claude Code and Codex CLI are pattern examples rather than the main subject. Claude Code gets one deeper concrete example so the reader can see the operating pattern in practice. The document must not become product documentation.

## Protected Boundary

Everything captured in `01-protected-prefix.md` is frozen. New writing starts after that prefix.

## Phase 2 Targets

- terminal-native agent environments
- core operating patterns
- artifacts as control surfaces
- skills and reusable workflows
- delegation and parallel subagents
- verification, recovery, and failure modes
- how to design with these patterns

## Explicit Exclusions

- step-by-step build tutorial
- framework shopping
- provider or model comparison
- broad agentic AI theory not directly needed for local agent environments
